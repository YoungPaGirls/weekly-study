# weekly-study
> 일주일 동안 하고싶은 공부하는 스터디 🍀

* 기간: 2023년 02월 01일 ~

* 스터디 규칙
  
  * 매주 이슈 템플릿에 맞춰 주간 목표를 올린다.
  
  * 달성하면 작성한 TODO를 체크하고 완료한 날짜를 기재한다.

* 목적 및 목표
  
  * 어떤 일이든 매주 목표를 세우고 달성하기
  
  * 개인의 상황에 맞춰 분량을 조절하기
  
  * 목표한 일들을 모두 하지 못하게 되어도 끝까지 최선을 다하기
  
  * 가장 중요한 것! 끝까지 포기하지 않기 👀✨
